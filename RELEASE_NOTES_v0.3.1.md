# Cheese Stick Dock Release Notes

제품 소개 문구: **Cheese Stick Dock**은 치지직 스트리머를 위한 설정 관리 & 방송 통계 독(Dock) 애플리케이션입니다.

## v0.3.1

배포일: 2026-04-28

### 변경 사항
- 로그인 화면의 개인정보 안내 문구 위치를 버튼 아래로 이동했습니다.
- 안내 문구를 최신 정책 표현으로 정리해 가독성을 높였습니다.
- 안내 영역의 박스 강조 스타일(배경/테두리)을 제거해 UI 톤을 자연스럽게 조정했습니다.

### 사용자 영향
- 계정 연동 전 화면에서 안내 문구가 덜 과하게 보이며, 메시지 이해가 쉬워졌습니다.
- 기능 동작이나 인증 플로우에는 변경이 없습니다.

### 포함 커밋
- `48a81eb` fix: 인증 안내 문구 위치와 스타일 완화
